Hello,

This is an exercise from Tarrega. It's my first tab posting,
so I hope I didn't make a lot of mistakes. The piece is in
some points fingerbreaker, though the rest of it is easily
performed when one can make quick chord changes.

[measure 8 corrected - 11 march 2005 - Weed - weed@wussu.com]

If there are people out there interested in more Tarrega stuff,
I would try to find some time (it really takes a lot of time
to write this stuff down) to write more pieces.

Have a lot of fun.

Dimitris

P.S.Any classic music tab out there?

Notation:
In some critical points left-hand fingering is indicated.
Right-hand fingering: p=thumb, i=index, m=middle, a=annular(ring)
Cxx where xx is a roman number, means fret with the first finger
the xx fret (in most the cases not all the chords).

[Estudio 20 in A major]
