this is a fantastic bossa-nova tune transcribed here for solo guitar.
it's likely you've heard a different version of the song as there are
many arrangements out there.  this version adheres to Gerald Garcia's
recording of the piece (which is the only score i've seen).

for those newer to LHF on guitar:
  1  index finger
  2  middle finger
  3  ring finger
  4  little finger

oh, and also
  /  slide up
  \  slide down
  h  hammer on
  p  pull off
  |  beat counts

other stuff you see littering the tab is probably my form directions.

as for dave's tab here, i don't know where he got the
--8p7---5-- part in the intro from, i don't hear it.
and the last chord is just a Gmaj7, think this is how
it goes:

--9/10-10-10\\---
-11/12----12\\---

one more thing which sounds great is to use hammer ons
on the sixth string for the bass line, ie.
--0h7-- and also possibly --0h5-- for those parts in
the main section.  slava uses hammers the 7th frets but
not the 5th frets (dave says he uses open A's instead
for some strange reason)..but both sound great.
